Updates the state by listening for the state relevant Keys i.e. Escape and Enter. Exact behaviour depends on current MenuState but default is: If Enter is pressed the game restarts at current level. If Escape is pressed the game quits.

Parameters
viewThe view to be updated.
quitIf set to true the game ends.
